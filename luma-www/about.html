About Luma
Luma is a graphical utility for accessing and managing data stored on LDAP servers. It is written in Python, using PyQt and python-ldap. Plugin-support is included and useful widgets with LDAP-functionality for easy creation of plugins are delivered.
Currently Luma supports english, german, portuguese and norwegian.


The following plugins are available:
  • Browser
  • Schema browser
  • Search
  • Addressbook
  • Template
  • Massive User Creation
  • Admin Utilities


Language support
Since Luma first started up, Luma has gained more and more users internationally. And more and more volounteer to translate Luma into their own native languange. For the latest release Luma support the following languages:


  • English
  • German
  • Norwegian (bokmål)
  • Swedish
  • Russian
  • Portoguese (Brazil)
  • French

More translators are welcome. Specially volounteers for Norwegian (nynorsk) are appreciated.
