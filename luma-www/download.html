Requirements
In order to run Luma, you need the following software:
  • Python (>=2.3): www.python.org
  • Qt (>=3.2): www.trolltech.com
  • PyQt (>=3.10): http://www.riverbankcomputing.co.uk/pyqt/
  • python-ldap (>=2.0.1): http://python-ldap.sourceforge.net/

To use NT and LANMAN passwords, the python smbpasswd module is needed: http://barryp.org/software/py-smbpasswd
